# Resolve Threshold-Outcome mismatch anti-pattern

## Raw Idea

فیلد 'integrity' برای برخی بسته‌ها (مانند accepts) وجود دارد، اما برای برخی دیگر (مانند array-flatten) وجود ندارد. این ناهماهنگی می‌تواند باعث شود که یکپارچگی برخی وابستگی‌ها تأیید نشود و امنیت پروژه به خطر بیفتد.
