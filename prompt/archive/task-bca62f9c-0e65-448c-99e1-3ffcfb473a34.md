## 💡 نمونه‌های قبل/بعد

_قبل:_
```
const wss = new WebSocketServer({ server, path: '/ws/live' });

app.use(cors());
```

_بعد:_
```
const wss = new WebSocketServer({ server, path: '/ws/live' });

wss.on('connection', (ws, req) => {
  console.log('WebSocket client connected');
  ws.on('message', (data) => {
    // Handle audio data or text messages
  });
  ws.on('close', () => {
    console.log('WebSocket client disconnected');
  });
});

app.use(cors());
```
