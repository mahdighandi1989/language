## 💡 نمونه‌های قبل/بعد
**حذف hardcoded config از HTML**

_قبل:_
```
var __firebase_config = JSON.stringify({ apiKey: "...", ... });
```

_بعد:_
```
// Firebase config از متغیرهای محیطی خوانده می‌شود
import.meta.env.VITE_FIREBASE_API_KEY
```
